二进制位反转思路

以下二进制位用8位做演示

a = 0000 0110 ; 十进制： 6

反转二进制位后

a = 0000 0011 ; 十进制： 3

只保留一个数的最低位，其余位变为0：
n = 0000 1101;
n & 1 = 0000 0001;

假设a = 0000 0110,b=0(用来保存结果)

a每次右移一位，然后最低位与b的最低位 进行位或，然后b左移一位， 直到a == 0为止，这样 b中则是a的反转二进制位的。

过程演示

  1. a = 0000 0110
    b = 0000 0000
  2. a = 0000 0011

    b = 0000 0001
  3. a = 0000 0001

    b = 0000 0011
  4. a = 0000 0000 停止，此时b就是a反转后的结果
